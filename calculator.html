Objective

The goal was to create a fully functional and aesthetic to-do list using HTML, CSS, and JavaScript.

Description

A fully functional to-do list made to increase my JavaScript knowledge.
When there is no category created, the create button will intermittently change color between white and yellow, when a category is created the button will turn white. When creating a category you will be able to choose its name and add tasks to it, you can mark tasks as completed or you can remove them, if you don't want that category after all, you can simply remove it.
